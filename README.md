# 通用图像识别
可以用来对任何图像项目进行识别

## 项目介绍   
一个基于tensorflow框架的Yolo小项目

## 项目配置  
* 作者开发环境：  
* Python 3.7  
* tensorflow 2.0
* opencv-python
